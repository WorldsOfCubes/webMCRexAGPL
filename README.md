## Information

 CMS with scripts for realize "online-mode" in Minecraft

### Base functionality

 - Admin panel with RCON console
 - Server monitoring system (RCON, Query, Simple Query)
 - Skin / Cloak uploader (default skin for male / female character)
 - Autodonate system
